# 🛒 Food Cart Web Application

A lightweight, interactive frontend food cart website. This project demonstrates responsive web design and dynamic UI features using vanilla web technologies.

## 🚀 Features
* **Dynamic Shopping Cart:** Add items, remove items, and update quantities with real-time price calculations.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.
* **Interactive UI:** Smooth transitions, hover effects, and modern aesthetic.

## 🛠️ Technologies Used
* **HTML5** - For semantic structure and layout.
* **CSS3** - For custom styling, Flexbox/Grid, and responsiveness.
* **JavaScript (ES6+)** - For DOM manipulation, event handling, and cart logic.
